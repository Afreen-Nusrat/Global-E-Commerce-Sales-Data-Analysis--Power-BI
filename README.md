# Global-E-Commerce-Sales-Data-Analysis--Power-BI
This project analyzes global e-commerce sales using Power BI. The data was cleaned using Power Query, and basic DAX measures were created to calculate total of sales, orders, and quantity. Interactive dashboards were built to analyze sales by country, category, and month. The project focuses on presenting data in a clear an easy-to-understand way. 
# Objective
- Analyze overall sales and revenue
- Identify top-performing countries and product categories
- Understand sales trends over time
# Dataset Description
The dataset includes:
- Order details
- Country and region
- Product category
- Sales and profit values
# Tools & Technologies Used
- Power BI
- Power Query
- DAX (basic measures)
- Microsoft Excel
# Key Insights
- Sales performance varies significantly across regions
- Certain product categories contribute more to total revenue
- Monthly sales trends help identify peak performance periods
# Project Output
- Interactive Power BI dashboard showing sales by country, category, and time period
# Conclusion
This project demonstrates the use of Power BI for data cleaning, analysis, and visualization to support data-driven decision making.
